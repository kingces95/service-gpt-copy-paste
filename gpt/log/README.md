# Interaction Log

This directory holds terse running summaries of interactions between Chris and
AI assistants.

Use one file per month. Prefer short dated entries over narrative detail.

Good entries capture:

- durable design decisions
- new roots or mappings
- package additions
- repo memory/doc updates
- changes in collaboration policy

Avoid transcripts. If a note needs detail, link or point to the relevant
`gpt/`, `docs/`, or source file.
