# Style Notes

This directory tracks design preferences and habits that make an idea feel
"Chris-ish" in this repo.

These notes should help future AI sessions make suggestions that fit the local
taste of the project rather than merely solving the immediate problem.

## Notes

- [Assistant Practices](assistant-practices.style.md): collaboration and editing
  habits for future assistants.
- [Chris-ish](chris-ish.style.md): vocabulary and design preferences that make
  suggestions fit the project.
