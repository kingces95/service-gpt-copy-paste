# Note Policy

## Commit Messages

GitHub does not render Markdown inside commit messages. Long fenced examples
and tree diagrams are harder to read there, and special characters may degrade.

Prefer commit messages that are plain text:

```txt
Short subject

Terse summary of the change and why it matters.

Detailed design note:
https://github.com/.../gpt/notes/YYYY-MM-DD-###-topic.notes.md
```

The commit message should orient the reader. The linked note should carry the
rendered design story, examples, trees, and longer rationale.

## Notes

Use `gpt/notes` for durable architectural turns, naming decisions, and
checkin-scale summaries that deserve rendered Markdown.

Note filenames use:

```txt
YYYY-MM-DD-###-slug.notes.md
```

The `###` is a same-day sequence number. It keeps files from the same day
stable in editor tabs and directory listings without using hash-like names.

Examples:

```txt
2026-05-20-001-shape-to-probe.notes.md
2026-05-20-002-stl-concepts-as-shapes.notes.md
2026-05-20-003-partial-shape-design.notes.md
```

Use the next available number for the day. Do not renumber older notes after
links have been published.

Other GPT docs use a role suffix so editor tabs identify the document kind
without requiring the full path:

```txt
gpt/commits/*.commit.md
gpt/log/*.log.md
gpt/marketing/*.marketing.md
gpt/marketing/idioms/*.idiom.md
gpt/meta-prototype-chain/**/*.meta.md
gpt/partial/**/*.partial.md
gpt/quests/*.quest.md
gpt/roots/**/*.root.md
gpt/sessions/*.session.md
gpt/style/*.style.md
```

Keep `README.md` unsuffixed. It is the directory front door, and GitHub/editor
tools already treat that name specially.

When a commit introduces a notable idea:

- add or update a note before the commit
- link the note from the commit message
- link related notes to each other so the design trail is walkable

## Commit Notes Directory

The older `gpt/commits` directory can still hold commit-specific rendered
notes. Going forward, prefer `gpt/notes` when the content is a reusable design
note rather than a single-commit artifact.
